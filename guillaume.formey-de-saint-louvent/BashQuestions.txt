Bash :

Est-on obligé de travailler en ligne de commande ?
Possibilité de faire des scripts

Comment insérer un script ?
$ ./SCRIPTNAME.sh 

Comment commencer un script ?
Tout script commence par : #!/bin/bash

Comment insérer un commentaire ?
Mettre un # en début de ligne

Comment définir des variables ? Attribuer, déclarer une variable ?
message='Bonjour tout le monde.'
la variable a pour nom "message" et pour valeur "Bonjour tout le monde."
On créé une variable dans un script de la même manière que vu précédemment

Comment modifier une variable ?
On déclare une variable
var="mype"
On affiche la valeur de la variable
echo "$var"

On modifie la variable
var="lami20j"
On affiche la nouvelle valeur de la variable
echo "$var"
